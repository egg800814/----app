# 尾牙抽獎轉盤系統使用手冊

這是一套專為公司尾牙設計的雙螢幕互動抽獎程式。系統分為「大螢幕顯示端」與「控制台操作端」，支援自定義獎項、名單、音效與互動式選人功能。

## 📂 檔案架構與資料夾說明

在執行程式前，請確保以下資料夾結構完整，並將對應的素材放入指定位置：

```text
抽獎轉盤app/
├── program/                 # 程式核心代碼
│   ├── main.py              # 啟動入口
│   ├── data.json            # [自動生成] 儲存獎項與名單設定
│   └── ...
├── assets/                  # 素材資料夾 (請將圖片與音效放此)
│   ├── images/              # 圖片素材
│   │   ├── logo.png         # 預設轉盤中心 LOGO
│   │   ├── background.jpg   # 大螢幕背景圖
│   │   └── ...
├── assets/                  # 素材資料夾 (請將圖片與音效放此)
│   ├── images/              # 圖片素材
│   │   ├── logo.png         # 預設轉盤中心 LOGO
│   │   ├── background.jpg   # 大螢幕背景圖
│   │   └── ...
│   ├── presenters/          # [重要] 候選人照片區
│   │   ├── 總經理.jpg        # 檔名即為顯示名稱 (支援 .jpg, .png)
│   │   ├── 副總.png
│   │   └── ...
│   └── sounds/              # 音效素材
│       ├── drumroll.wav     # 轉動音效 (選用)
│       ├── win2.wav         # 中獎音效 (支援 win2.wav 自動偵測)
│       └── ...
└── ...
```

---

## 🚀 如何啟動

1.  確認 **雙螢幕** 已連接並設定為「延伸模式」。
2.  執行 `program/main.py`。
3.  程式會自動將 **大螢幕視窗** 投放到第二個螢幕，並在主螢幕顯示 **控制台視窗**。

---

## 🎮 控制台操作說明

控制台主要分為左側的「獎項設定」與右側的「轉盤預覽」。

### 1. 獎項與名單管理 (左側)
*   **選擇獎項**：從下拉選單選擇要抽的獎項（如：特獎、頭獎...）。
*   **編輯獎項**：可新增、修改或刪除獎項名稱。
*   **名單管理**：
    *   在文字框內直接貼上名單（一行一個名字）。
    *   點擊 **[💾 儲存並更新名單]** 才會生效。
    *   點擊 **[🔀 打散名單排序]** 可隨機洗牌。

### 2. 設定抽獎人頭像 (轉盤中心圖片)
有兩種方式可以設定轉盤中心的圖片：

#### A. 從電腦選擇 (後台指定)
*   點擊 **[📷 從電腦選擇照片]**。
*   選擇一張圖片檔案。
*   **效果**：圖片會以 **「完整比例 (Fit)」** 縮放至轉盤中心 (適合全身照或LOGO)。

#### B. 開啟大螢幕選人 (前台互動)
*   點擊 **[🖥️ 開啟大螢幕選人模式]**。
*   大螢幕會跳出 **相片牆 (Photo Grid)**，顯示 `assets/presenters` 資料夾內的所有照片。
*   **互動方式**：使用滑鼠（或觸控）點擊想要的人選（會有槌子敲擊效果）。
*   **效果**：圖片會自動 **「智慧裁切 (Smart Zoom)」**，將全身照裁切為 **半身/大頭特寫**，並同步顯示於轉盤中心。

### 3. 微調轉盤手感 (右側下)
*   **滑行阻力 (Friction)**：數值越低滑得越久，越高停得越快。
*   **煞車力道 (Brake)**：接近停止時的減速力道。
*   **調整方式**：拖拉滑桿後，點擊 **[測試轉動手感]** 預覽效果。

### 4. 執行抽獎
1.  點擊 **[發布到大螢幕]**：確認目前的獎項、名單、頭像都已設定好。
2.  點擊 **[同步啟動抽獎]** (系統端) 或在大螢幕長按 **[開始抽獎]** 按鈕。
3.  **中獎流程**：
    *   轉盤停止，大螢幕顯示中獎動畫。
    *   系統端跳出確認視窗。
    *   **確認 (Confirm)**：名單自動刪除該中與者，並記錄到結果。
    *   **保留 (Cancel)**：不刪除名單，此局不算或重抽。

## 🖼️ 素材準備注意事項

### 1. 候選人照片 (`assets/presenters`)
*   **檔名**：請直接使用人名或職稱命名 (例如 `王小明.jpg`、`董事長.png`)，這將方便管理。
*   **格式**：建議使用直式長方形或正方形照片 (JPG/PNG)。
*   **自動裁切**：程式內建智慧裁切功能。
    *   若想在大螢幕選人時顯示清晰大頭照，請準備 **全身或半身照** 即可，程式會自動放大並取上半部。
    *   若照片本身已經是特寫大頭照，顯示效果會更近。

### 2. 中心預設 LOGO (`assets/images/logo.png`)
*   若未選擇任何抽獎人，轉盤中心預設會顯示此圖。
*   建議使用去背的 PNG 檔案，效果最佳。

### 3. 音效 (`assets/sounds`)
*   **win2.wav**：若存在此檔案，中獎時會優先播放並覆蓋其他音效。
*   請確保音量適中。

## ⚠️ 常見問題

*   **Q: 大螢幕選完照片後，系統端預覽沒有變？**
    *   A: 請確認程式是否為最新版本。正常情況下，系統端會同步更新為「特寫模式」的預覽圖。

*   **Q: 轉盤轉太久停不下來？**
    *   A: 請在控制台右下角調高「滑行阻力」數值，然後重新測試。

*   **Q: 如何重置所有設定？**
    *   A: 關閉程式，刪除 `program/data.json` 檔案，重新啟動程式即可恢復預設值。
